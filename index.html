Try Work in Progress

Type in the first few characters of what you are looking for and let the search box do the work for you. Select a match using either the mouse or the up/down arrow keys and Enter.

Address: Enter the first part of the address. Ex: for 700 North Tryon Street, try 700 n

Place: Enter any part of a place name. Ex: type carolina

Street: Enter the name of the street. Ex: type ruth

Intersection: Enter a street name, followed by an &. Ex: type ruth &

Parcel ID: Enter an 8-digit Parcel ID. Ex: type 11111111

Your Current Location: If your web browser supports geolocation, you will be prompted to allow its use and you will see Find My Location under the search bar.
