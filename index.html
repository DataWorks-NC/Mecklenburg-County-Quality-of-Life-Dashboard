The secrect to successful searches is to let the search box do the work for you. Select a match using either the mouse or the up/down arrow keys and Enter.

Address: Enter the first part of the address. Ex: for 700 North Tryon Street, try 700 n

Place: Enter any part of a place name. Ex: type carolina

Street: Enter the name of the street. Ex: type ruth
